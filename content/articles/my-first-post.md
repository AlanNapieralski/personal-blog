# My First Post

This is my first blog post using the new markdown-based content management system!

## Why Markdown?

Markdown is simple, portable, and perfect for writing content. No need for a heavy backend like Drupal.

## Benefits

- Easy to write
- Version controlled
- No database needed
- Fast and lightweight
